# Custom Token Farm

A demo for staking LP Token to farm Native Token.

# Documentation

- [Live Demo](https://cmt-lp-farm.netlify.app/)
- [Development](#development)

# Development

## Initial Setup

##### Install all dependencies for React App

```bash
npm: npm install
yarn: yarn install
```

## Running App

##### Running app in development enviroment

```bash
npm: npm run dev
yarn: yarn dev
```
